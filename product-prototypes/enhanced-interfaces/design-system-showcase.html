Implementation Guide

Code examples for integrating with your Next.js + TailwindCSS project.

Theme Implementation

// Apply theme via data attribute document.body.dataset.theme = 'classic'; // Or use class-based approach document.body.className = 'theme-classic'; // Theme toggle function function switchTheme(themeName) { document.body.dataset.theme = themeName; localStorage.setItem('coffee-theme', themeName); }

Component Usage

<button class="btn btn--primary btn--large"> <svg class="icon icon--sm">...</svg> Start Roast </button> <div class="timer-display timer-display--active"> <div class="timer-display__time">12:45</div> <div class="timer-display__label">Roast Time</div> </div>

// Focus management element.setAttribute('aria-live', 'polite'); element.setAttribute('aria-describedby', 'help-text'); // Screen reader support <button aria-label="Emergency stop roast session"> Emergency Stop </button>
