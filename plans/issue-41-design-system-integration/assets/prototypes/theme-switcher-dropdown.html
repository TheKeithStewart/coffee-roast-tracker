Design System Theme Demonstration

This interactive prototype demonstrates the Coffee Roast Tracker design system with four distinct color themes. Use the theme switcher in the navigation to see how all components adapt seamlessly to your preference.

Each theme is carefully designed to meet WCAG 2.1 AA accessibility standards while providing a unique user experience optimized for different lighting conditions and user preferences.
